## Get started in minutes

<Steps>
  <Step title="Start Higress with Docker">
    The fastest way to run Higress locally is with a single `docker run` command:

    ```bash
    mkdir higress && cd higress
    docker run -d --rm --name higress-ai \
      -v ${PWD}:/data \
      -p 8001:8001 -p 8080:8080 -p 8443:8443 \
      higress-registry.cn-hangzhou.cr.aliyuncs.com/higress/all-in-one:latest
    ```

    - **8001** — Higress UI console
    - **8080** — Gateway HTTP entry
    - **8443** — Gateway HTTPS entry
  </Step>
  <Step title="Open the console">
    Navigate to `http://localhost:8001` to access the Higress UI console where you can configure routes, plugins, domains, and certificates.
  </Step>
  <Step title="Configure an AI route">
    Add the `ai-proxy` plugin to a route and point it at your preferred LLM provider. Higress normalizes all providers to the OpenAI API format.

    See the [AI Proxy Guide](/guides/ai-proxy) for a step-by-step walkthrough.
  </Step>
  <Step title="Extend with plugins">
    Browse the [Plugin Reference](/plugins/overview) to add rate limiting, authentication, WAF protection, and more — no restarts required.
  </Step>
</Steps>
